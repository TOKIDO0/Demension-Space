# 在 Cursor 中打开两个预览窗口的指南

## 方法 1: 使用我创建的脚本（推荐）

我已经为您创建了一个脚本 `open-preview.js`，可以同时启动前台和后台预览服务器。

### 使用步骤：

1. **在 Cursor 终端中运行**：
   ```bash
   npm run preview
   ```
   或者直接运行：
   ```bash
   node open-preview.js
   ```

2. **脚本会自动**：
   - 启动前台服务器（端口 3000）：`http://localhost:3000`
   - 启动后台服务器（端口 3001）：`http://localhost:3001`
   - 自动在浏览器中打开两个窗口

## 方法 2: 使用 Live Server 扩展（最简单）

如果您安装了 **Live Server** 扩展：

1. **打开前台页面**：
   - 在文件资源管理器中，右键点击 `public/index.html`
   - 选择 "Open with Live Server"
   - 会在默认浏览器中打开前台页面

2. **打开后台页面**：
   - 在文件资源管理器中，右键点击 `public/admin/index.html`
   - 选择 "Open with Live Server"
   - 会在新标签页中打开后台页面

## 方法 3: 手动在浏览器中打开

1. **前台页面**：
   - 在 Cursor 中打开 `public/index.html`
   - 按 `Ctrl+Shift+P`（Mac: `Cmd+Shift+P`）打开命令面板
   - 输入 "Simple Browser: Show" 或 "Preview"
   - 选择预览命令

2. **后台页面**：
   - 在 Cursor 中打开 `public/admin/index.html`
   - 重复上述步骤，会在新窗口中打开

## 方法 4: 使用 Python 简单服务器（如果已安装 Python）

### 打开两个终端窗口：

**终端 1 - 前台服务器**：
```bash
cd public
python -m http.server 3000
```
然后在浏览器中打开：`http://localhost:3000`

**终端 2 - 后台服务器**：
```bash
cd public/admin
python -m http.server 3001
```
然后在浏览器中打开：`http://localhost:3001`

## 方法 5: 使用 VS Code/Cursor 的预览功能

1. **打开第一个预览**：
   - 打开 `public/index.html`
   - 按 `Ctrl+K V`（Mac: `Cmd+K V`）打开侧边预览
   - 或者右键点击文件，选择 "Open Preview"

2. **打开第二个预览**：
   - 打开 `public/admin/index.html`
   - 按 `Ctrl+K V` 打开另一个预览窗口
   - 两个预览会并排显示

## 推荐配置

我建议使用 **方法 1（脚本）** 或 **方法 2（Live Server）**，因为它们最简单且自动化程度高。

### 安装 Live Server 扩展（如果还没有）：

1. 在 Cursor 中按 `Ctrl+Shift+X`（Mac: `Cmd+Shift+X`）打开扩展面板
2. 搜索 "Live Server"
3. 安装 "Live Server" by Ritwick Dey
4. 安装后，右键点击 HTML 文件即可看到 "Open with Live Server" 选项

## 注意事项

- 如果端口被占用，脚本会自动使用已运行的服务器
- 确保两个页面使用不同的端口，避免冲突
- 前台页面：`http://localhost:3000`
- 后台页面：`http://localhost:3001`
